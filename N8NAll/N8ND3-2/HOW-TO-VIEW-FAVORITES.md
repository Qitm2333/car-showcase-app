# 📖 如何查看收藏？完整指南

## 🎯 三种查看方式

### 方式 1：查看所有收藏（推荐）👍

1. **点击顶部导航栏的 "💖 我的收藏" 按钮**
2. 你会看到所有收藏，**按文件夹分组显示**
3. 每个文件夹显示标题和收藏数量
4. 可以直接浏览和删除收藏

```
💖 我的收藏
━━━━━━━━━━━━━━━━━━
📂 收藏夹 → 所有收藏

📁 默认收藏夹 (3)
┌────┐ ┌────┐ ┌────┐
│车1 │ │车2 │ │车3 │
└────┘ └────┘ └────┘

💖 我的最爱 (2)              [查看全部 →]
┌────┐ ┌────┐
│车4 │ │车5 │
└────┘ └────┘

🚗 豪华车系 (1)              [查看全部 →]
┌────┐
│车6 │
└────┘
```

---

### 方式 2：通过收藏夹管理页面 📂

1. **点击顶部导航栏的 "📂 收藏夹" 按钮**
2. 查看所有收藏夹列表
3. **点击任意收藏夹卡片**
4. 进入该收藏夹，只显示该文件夹的收藏

```
📂 收藏夹管理
━━━━━━━━━━━━━━━━━━
┌─────────┐  ┌─────────┐
│   💖    │  │   🚗    │
│ 我的最爱 │  │ 豪华车系 │  ← 点击进入
│   (2)   │  │   (1)   │
└─────────┘  └─────────┘

点击后 ↓

💖 我的最爱
━━━━━━━━━━━━━━━━━━
📂 收藏夹 → 💖 我的最爱

┌────┐ ┌────┐
│车4 │ │车5 │
└────┘ └────┘
```

---

### 方式 3：直接从收藏页面跳转 🎨

1. 在 "🎨 去收藏" 页面收藏车辆后
2. 点击 "💖 我的收藏" 按钮
3. 立即查看刚才的收藏

## 🚀 页面功能详解

### "💖 我的收藏" 页面功能

#### 1. 面包屑导航
```
📂 收藏夹 → 所有收藏
```
- 点击 "📂 收藏夹" 返回收藏夹管理
- 显示当前查看的是哪个文件夹

#### 2. 分组显示（查看所有时）
- 按文件夹自动分组
- 显示每个文件夹的收藏数量
- 点击 "查看全部 →" 进入单个文件夹

#### 3. 收藏卡片
- 🖼️ 车辆图片（悬停放大）
- 📝 车辆名称
- 🔖 车辆ID
- 📅 收藏时间
- 🗑️ 删除按钮（悬停显示）

#### 4. 操作按钮
- **[查看所有]** - 从单个文件夹返回所有收藏
- **[继续收藏 →]** - 跳转到收藏页面添加更多

## 📂 "收藏夹管理" 页面功能

### 功能列表
1. ✅ 查看所有收藏夹
2. ✅ 创建新收藏夹
3. ✅ 点击收藏夹查看内容
4. ✅ 删除收藏夹（悬停显示删除按钮）

### 创建收藏夹
1. 点击 **[➕ 创建新收藏夹]**
2. 输入名称（如"我的最爱"）
3. 选择图标（24个emoji可选）
4. 点击 **[创建]**

## 🔄 完整使用流程

### 场景 1：第一次使用

```
1. 登录系统
   ↓
2. 配置 API（在登录页或设置页）
   ↓
3. 点击 "📂 收藏夹" 创建第一个文件夹
   ↓
4. 点击 "🎨 去收藏" 添加收藏
   ↓
5. 点击 "💖 我的收藏" 查看
```

### 场景 2：日常使用

```
1. 登录系统
   ↓
2. 点击 "🎨 去收藏" 浏览车辆
   ↓
3. 点击喜欢的车辆收藏
   ↓
4. 点击 "💖 我的收藏" 查看所有
   ↓
5. 点击文件夹名称查看分类收藏
   ↓
6. 悬停在收藏上点击 🗑️ 删除
```

### 场景 3：管理收藏夹

```
1. 点击 "📂 收藏夹"
   ↓
2. 查看所有文件夹
   ↓
3. 点击 "➕ 创建新收藏夹" 添加分类
   ↓
4. 点击文件夹卡片查看该分类的收藏
   ↓
5. 悬停在文件夹上点击 🗑️ 删除（不会删除收藏）
```

## ⚠️ 常见问题

### Q1: 为什么看不到收藏？

**可能原因：**
1. ❌ 还没有添加任何收藏
2. ❌ 在"收藏夹管理"页面而不是"我的收藏"页面
3. ❌ API 未配置或配置错误
4. ❌ N8N Workflow 未激活

**解决方法：**
- 点击顶部 **"💖 我的收藏"** 按钮（不是"📂 收藏夹"）
- 检查 N8N Workflow 是否激活
- 在设置页面检查 API 配置

### Q2: 收藏夹和收藏有什么区别？

- **📂 收藏夹** = 文件夹（容器），用来分类
- **💖 收藏** = 具体的车辆（内容），保存在文件夹里

就像：
- 收藏夹 = 相册
- 收藏 = 照片

### Q3: 删除收藏夹会删除收藏吗？

**不会！** 删除收藏夹只是删除分类，收藏本身还在。

删除收藏夹后：
- ✅ 收藏保留
- ✅ 收藏自动归入"默认收藏夹"
- ✅ 可以在"💖 我的收藏"中查看

### Q4: 如何把收藏分类到不同文件夹？

**当前版本：**
- 所有收藏都保存到"默认收藏夹"

**下一版本（即将实现）：**
- 收藏时可以选择文件夹
- 可以移动现有收藏到其他文件夹
